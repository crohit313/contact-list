This Project is built on Angular version 7.

1. Install Angular CLI on the system.
2. open the project folder in cmd and install npm by following command:-
	npm install
3. now type Following Command to run the application :-
	ng serve
4. application will get open on http://localhost:4200
5. To run the test cases type following command:-
	ng test
